## Configuration & Setup

### Project Cloning

```sh
git clone https://github.com/rhea-so/BaekjoonTypescriptTemplate
cd BaekjoonTypescriptTemplate
npm config set registry https://npm.rhea-so.com/
npm i
```

## Build 

```sh
tsc
```

## Test

```sh
npm test
```

### Get Test Coverage

```sh
npm run coverage
```
